TIPER

A website for typing. Simple as that. Visit the Github here.

CLASSIC

To Play:
Click the play button below to start the game. Start typing immediately, and press the restart button at any time. If you make an error, the box will turn red, but backspacing will clear the error. When you type the full text, the timer will stop and you can play again. This game requires a physical keyboard
× This game makes use of keyboard focus on an element with the tabindex attribute. Unfortunately, on a phone or other device without a physical keyboard, the virtual keyboard will not pop up for you to play the game with when you start it.
