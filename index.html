CLASSIC

To Play:

Click the play or restart butons below to start, pause, and reset the game. You can also press the enter key at any time to start or resume a game. Once in a game, type the text as you normally would, and backspace if you type the wrong key. Good Luck! This game requires a physical keyboard

× This game makes use of keyboard focus on an element with the tabindex attribute. Unfortunately, on a phone or other device without a physical keyboard, the virtual keyboard will not pop up for you to play the game with when you start it.
